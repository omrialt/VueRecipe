# VueRecipe

> webapp to uplaod and share recipes, bulid with Vue and Node.js


## Features

- Full wishlist
- Recipe reviews and ratings
- Recipe search feature
- Recipe screen with similar recipes suggestions in the bottom
- Recipe add,edit and delete
